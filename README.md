##install
* on Windows 
  * if fail to start tkinter
    * python version 3.7 and up will include new tk
* on macOS 
  * if fail to start tkinter 
    * xcode-select --install
      * download python-tk
        * brew install python-tk
    * or download new python version, new python version include new Tcl/Tk
