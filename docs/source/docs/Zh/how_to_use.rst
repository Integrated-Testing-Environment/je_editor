如何使用
----

* JEditor 是一個簡單的文字編輯器，但具有所有應有的功能。
* JEditor 是 ITE 的構成組件之一，但元件化使得他可以單獨使用。
* JEditor 預設使用深色介面 如下所示:
.. image:: image/JEditor.png
* JEditor 有以下功能:
    * 字體改變。
    * 字體大小改變。
    * 自動儲存 (儲存與開啟後)。
    * 樹狀檔案結構查詢。
    * 儲存與開啟檔案。
    * Python 自動優化，使用 yapf。
    * 運行 Python 程式。
    * 運行 Shell command。
